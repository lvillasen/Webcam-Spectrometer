Explore the Visible Spectrum with a Webcam

CIIEC-BUAP

October 25, 2025

Source Code

This web application demonstrates how to build a simple, low-cost optical spectrometer using a manually focused commercial webcam as the detector. The setup includes a 3D-printed enclosure that mounts directly onto the webcam, a narrow slit covered with a thin layer of wax paper for light diffusion, and a small piece of a DVD that serves as a diffraction grating.

The DVD fragment disperses the incoming light into its component wavelengths, allowing the webcam to capture the resulting spectrum. The 3D-printed enclosure ensures precise alignment between the slit, the grating, and the camera lens.

This application provides both the OpenSCAD source code and the STL file required to fabricate the spectrometer enclosure with a 3D printer. The assembled spectrometer is shown in the photograph above.

The design focuses on adapting a custom 3D-printed enclosure to a webcam. We selected the GETTTECH FHD 1080p 30 FPS webcam, which features manual focus and a 100-degree field of view. The enclosure includes a 1.2 cm × 1.2 cm window that allows the user to make a 0.5 mm × 0.8 cm slit at the optimal position for the best spectral results. The slit can be easily constructed using two small pieces of rigid plastic, such as material cut from old credit cards.

Inside the box, a 2 cm × 3 cm fragment of a DVD is mounted flat on one of the inner walls, as shown in the image below. For better results, remove the polycarbonate plastic layer from the DVD fragment. The DVD grooves are carefully aligned with the slit inside the enclosure. Incoming light strikes the DVD at a grazing-incidence angle, which simplifies the optical geometry while improving angular resolution.

As can be calculated using this web application—the same tool used to obtain the figure below—grazing incidence has the property that diffraction angles change very little when the incidence angle varies slightly.

The design also allows the enclosure to slide over the webcam, enabling fine adjustment of focus and alignment to achieve the best spectral resolution.



The enclosure box is modeled in the OpenSCAD code below.


th = 1.5;         // Thickness of walls (mm)
box_x = 20+th;       
box_y = 32.5+2*th;       
box_z = 130;       

// Rectangular window
rect_z = 40;       

// Window for slit
slot_x = 15;   
slot_y = 12;   
slot_th = th;   

// --- Rectangular box, window on YZ plane and window on XY plane ---
module box() {
    difference() {
        // External box
        cube([box_x, box_y, box_z]);

        // Internal hollow
        translate([th, th, th])
            cube([box_x - 2*th, box_y - 2*th, box_z - 2*th]);

        // --- Rectangular window on YZ plane
        translate([0, th, 2+th])
            cube([th, box_y-2*th, rect_z]);

        // --- Window for slit
        translate([(box_x - slot_x)/2, (box_y - slot_y)/2, box_z - slot_th])
            cube([slot_x, slot_y, slot_th]);
    }
    translate([-2*th,0,0]) cube([2*th,th,box_z]);
    translate([-2*th,box_y-th,0]) cube([2*th,th,130]);
}

box();

A test of the optical spectrometer described is presented here, using a compact fluorescent lamp (CFL) as the light source and this web application to acquire and to calibrate the webcam data.

Note that this spectrometer is calibrated only in wavelength, not in absolute intensity. Furthermore, the color webcam used has a Bayer filter that distorts the relative intensities of different wavelengths. A future version of the Spectrum2 web application will include intensity calibration to correct for the uneven spectral response of the webcam sensor.
